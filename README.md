# **Cygnus**

Rest API made in Nestjs, using some concepts of Clean Architecture and some Design Patterns

### Requisits to run this project

- Docker
- Docker-compose

### To run this project, follow the below steps

##### Build Project
   - `npm i`
   - `docker-compose up --build`
